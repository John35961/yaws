About

This website is a simple web application meant to provide you with live and accurate weather data for the location of your choice.

The data is pulled from:

  • OpenStreetMap Nominatim API to geocode user's input
  • TimeZoneDB API to get location's local time
  • OpenWeatherMap API for weather conditions and forecasts
  • AirVisual API for the air quality index

This website is powered by:

  • Python 3.6 for the language
  • Flask for the webframework running the application
  • Bootstrap 4 for the frontend UI
  • ChartJS to build beautiful weather data charts
  • AWS Elastic Beanstalk provides the platform on which the application is running
  • AWS CodePipeline handles the deploys

If you have any feedback or are experiencing troubles using the website Please feel free to contact us!

Thanks for using yaws !
